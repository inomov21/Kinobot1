# 🎬 Kino Bot

## O'rnatish

```bash
pip install -r requirements.txt
python bot.py
```

## Fayllar

- `bot.py` — Asosiy bot fayli
- `config.py` — Token va Admin ID
- `database.py` — Database (SQLite)
- `keyboards.py` — Barcha tugmalar
- `handlers/admin.py` — Admin panel
- `handlers/user.py` — Foydalanuvchi panel

## Buyruqlar

- `/start` — Botni ishga tushirish
- `/admin` — Admin panelini ochish

## Admin panel bo'limlari

1. 📊 Statistika
2. ✉️ Xabar yuborish
3. 🎬 Kontent boshqaruvi
   - 🎬 Kinolar (yuklash, tahrirlash, o'chirish)
   - 📌 Postlar
   - 🔗 Referal
4. 🔑 Kanallar (majburiy obuna)
5. ⚙️ Tizim sozlamalari
   - 📢 Reklama
   - 👮 Adminlar
   - 📝 Matnlar
6. 📩 So'rovlar
7. 👥 Foydalanuvchilar
8. 🏆 Top 10
9. 💬 Muhokama

- 🏆 Top 10
